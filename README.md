## Описание

В данной лабораторной работе было создано простое веб-приложение
**TODO-лист** для управления списком задач.

В ходе выполнения работы были изучены и закреплены следующие темы:

- работа с DOM (Document Object Model)
- поиск HTML-элементов через JavaScript
- создание элементов динамически
- обработка событий (click, keypress)
- работа с формами и полем ввода
- динамическое обновление интерфейса
- добавление, выполнение и удаление задач
- обновление счётчика задач
- базовая работа с Git и GitHub

Приложение позволяет:

- добавлять новые задачи
- отмечать задачи как выполненные
- удалять задачи
- видеть общее количество задач

## Основной функционал TODO

Приложение реализует следующий функционал:

1.  Добавление новой задачи
2.  Отображение списка задач
3.  Отметка задачи как выполненной
4.  Удаление задачи
5.  Подсчёт общего количества задач

Основные функции JavaScript:

- `addNewTask()` - добавляет новую задачу\
- `createTaskItem()` - создаёт DOM элемент задачи\
- `updateCounter()` - обновляет счётчик задач
